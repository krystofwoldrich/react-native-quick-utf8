# react-native-quick-utf8

Converts JS String to UTF-8 Encoded Byte Array and vice versa.

## Installation

```sh
npm install react-native-quick-utf8
```

## Usage

```js
import { multiply } from 'react-native-quick-utf8';

// ...

const result = await multiply(3, 7);
```
